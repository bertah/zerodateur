<h2>Installation</h2>

Download Python >= 3.5
https://www.python.org/downloads/

Install requirements using pip
> pip install -r "C:\path\to\requirements.txt"

Run the script and leave it running
> python zerodt.py

Note: Script will run for 7 days only, see todo
